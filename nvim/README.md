# Neovim Config

My personal minimal, focused Neovim setup built for clarity, speed, and intentional development.

No bloat.  
Just essential tooling.

---

## Goals

- Keep it lightweight  
- Add only what improves workflow  
- Prefer understanding over copying large configs  
- Let real usage guide configuration  

---

## Core Plugins

- **Telescope** – fast fuzzy finding  
- **Gitsigns** – inline Git diff awareness  
- **Lualine** – clean, minimal statusline  
- **LSP** – language intelligence
- **Treesitter** – Syntax highlighting

---

## Design Choices

- Per-window statusline  
- Relative file paths in statusline  
- Minimal separators  
- Git-aware workflow  
- Terminal-native aesthetic
